Help

On this page you can find a brief explanation about the different pages of this web application.

Database Search

This page can be used to search the in the results of the BLASTed sequences in the database.

Input
The input field accepts text and/or numbers. The application will search the database using the user input and return return results if there is a hit with the values in the following tables:

  • seq id
  • acc id
  • protein name
  • protein function
  • organism
  • genus
  • family
The parameter seq id refers to the sequence id, which is used in the database to connect the sequences to the BLAST results. The parameter acc id refers to the accession code used in NCBI. When the input field is left empty the application will return all the results.

BLAST

This page can be used to BLAST user inputed sequences using qblast.
A DNA sequence can be submitted in the grey lined box. Clicking the “blast” button results in an allignment of the sumbitted sequence against the nr-database.
The following parameters have been modified:

  • Database: nr
  • Algoritm: blastx
  • Hit List Size: 10
  • Expect Value: 1e-5
All the other parameters have the qblast default values (link: BioPython qblast documentation).
